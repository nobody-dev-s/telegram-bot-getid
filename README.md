## 使用方法
1.[申请](https://t.me/BotFather)一个telegram机器人,获取token<br>
2.新建telegram群组，邀请机器人进群，给予管理员权限<br>
3.安装[rust](https://rustup.rs/)<br>
4..env文件中配置，替换你的机器人 `token` 启动程序<br>

``` javascript
token="替换为你的机器人token"
```
